Vistas de Sesiones de PolyAcademy

Descripción General

Las vistas de sesiones proporcionan la funcionalidad necesaria para gestionar la autenticación y el seguimiento de sesiones de usuarios en la plataforma. Estas vistas permiten el inicio de sesión unificado para estudiantes y profesores, así como el registro y gestión de las sesiones.

Inicio de Sesión Unificado

Como usuario, necesito poder iniciar sesión en la plataforma con mis credenciales.

  • Autenticación por email y contraseña
  • Identificación automática del tipo de usuario
  • Generación de tokens de acceso
  • Validación de credenciales

Registro de Sesiones

Como administrador, necesito mantener un registro de las sesiones de los estudiantes.

  • Registro de fecha y hora de inicio de sesión
  • Filtrado por estudiante específico
  • Ordenamiento cronológico de registros
  • Historial completo de accesos

Cierre de Sesión

Como usuario, necesito poder cerrar mi sesión de forma segura.

  • Invalidación de tokens activos
  • Confirmación de cierre exitoso
  • Manejo de errores en el proceso
  • Protección de rutas autenticadas

Validación de Usuarios

Como sistema, necesito validar el tipo y permisos de cada usuario.

  • Verificación de roles (estudiante/profesor)
  • Validación de usuarios activos
  • Control de acceso basado en roles
  • Manejo de usuarios sin rol asignado

Características Técnicas

Detalles técnicos importantes de la implementación

  • Uso de JWT para autenticación
  • Respuestas estandarizadas con status
  • Documentación con drf-spectacular
  • Manejo robusto de errores

Beneficios Clave

  • Sistema de autenticación unificado
  • Gestión segura de sesiones
  • Trazabilidad de accesos
  • Control granular de permisos
  • Experiencia de usuario mejorada
