Vistas de Clases de PolyAcademy

Descripción General

Las vistas de clases proporcionan la funcionalidad necesaria para gestionar las clases dentro de los cursos de la plataforma. Estas vistas están diseñadas para satisfacer las necesidades tanto de profesores como de estudiantes.

ClassModelViewSet

Como profesor, necesito gestionar las clases de mis cursos de manera integral.

  • Listar Clases: Necesito ver todas las clases de un curso específico
  • Filtrado: Debo poder filtrar las clases por curso usando course_id
  • Crear Clase: Necesito crear nuevas clases asociadas a un curso específico
  • Actualizar Clase: Debo poder modificar la información de las clases existentes
  • Eliminar Clase: Necesito poder eliminar clases cuando sea necesario
  • Respuestas: Debo recibir confirmaciones claras de las operaciones realizadas

Endpoints Disponibles:

  • GET /api/classes/ - Lista todas las clases
  • GET /api/classes/?course_id={id} - Lista clases de un curso específico
  • POST /api/classes/ - Crea una nueva clase
  • PUT /api/classes/{id}/ - Actualiza una clase existente
  • DELETE /api/classes/{id}/ - Elimina una clase

ClassListView

Como estudiante o profesor, necesito ver la lista de clases de un curso específico.

  • Listado Filtrado: Ver solo las clases del curso seleccionado
  • Organización: Visualizar las clases en orden secuencial
  • Información Básica: Ver detalles esenciales de cada clase

Endpoint Disponible:

  • GET /api/courses/{course_id}/classes/ - Lista las clases de un curso

ClassDeleteView

Como profesor, necesito eliminar clases de manera segura y controlada.

  • Eliminación Segura: Confirmar la existencia de la clase antes de eliminar
  • Confirmación: Recibir confirmación de la eliminación exitosa
  • Manejo de Errores: Recibir mensajes claros si hay problemas
  • Datos de Retorno: Obtener información de la clase eliminada

Endpoint Disponible:

  • DELETE /api/classes/{id}/delete/ - Elimina una clase específica

Ejemplos de Uso

Crear una Nueva Clase


POST /api/classes/
{
    "class_name": "Introducción a la Gramática",
    "description": "Conceptos básicos de gramática inglesa",
    "course": 1,
    "bullet_points": ["Verbos", "Sustantivos", "Adjetivos"]
}

Listar Clases de un Curso


GET /api/classes/?course_id=1

Eliminar una Clase


DELETE /api/classes/1/delete/
