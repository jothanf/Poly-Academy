Vistas de Escenarios de PolyAcademy

Descripción General

Las vistas de escenarios proporcionan la funcionalidad necesaria para gestionar los escenarios de práctica conversacional en la plataforma. Estas vistas permiten a los profesores crear y gestionar escenarios interactivos para que los estudiantes practiquen sus habilidades de conversación.

Listar Escenarios

Como profesor, necesito ver una lista de todos los escenarios disponibles para una clase específica.

  • Filtrado automático por class_id
  • Ordenamiento por fecha de creación más reciente
  • Vista paginada de escenarios
  • Información resumida de cada escenario

Crear Escenario

Como profesor, necesito crear nuevos escenarios de práctica para mis estudiantes.

  • Formulario completo para datos del escenario
  • Soporte para carga de imágenes de portada
  • Validación de campos requeridos
  • Respuesta detallada de éxito/error

Actualizar Escenario

Como profesor, necesito modificar los escenarios existentes para mejorar su contenido.

  • Actualización parcial o completa de datos
  • Mantenimiento de archivos multimedia existentes
  • Validación de cambios propuestos
  • Confirmación de actualización exitosa

Eliminar Escenario

Como profesor, necesito eliminar escenarios que ya no sean relevantes.

  • Eliminación segura de datos
  • Limpieza de archivos multimedia asociados
  • Confirmación de eliminación exitosa
  • Manejo de errores en caso de fallos

Características Técnicas

Detalles técnicos importantes de la implementación

  • Soporte para múltiples formatos de archivo (MultiPartParser)
  • Procesamiento de datos JSON (JSONParser)
  • Manejo de formularios tradicionales (FormParser)
  • Respuestas estandarizadas con status y mensajes

Beneficios Clave

  • Gestión intuitiva de escenarios de práctica
  • Respuestas consistentes y descriptivas
  • Manejo robusto de errores
  • Soporte para contenido multimedia
  • Organización eficiente por clase
