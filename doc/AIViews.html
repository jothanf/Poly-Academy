Vistas de Inteligencia Artificial de PolyAcademy

Descripción General

Las vistas de IA de PolyAcademy proporcionan funcionalidades avanzadas de procesamiento de lenguaje natural, generación de imágenes y conversión de texto a voz para mejorar la experiencia de aprendizaje.

AskOpenAIView

Como usuario, necesito poder hacer preguntas al asistente de IA para recibir ayuda y orientación en mi aprendizaje.

  • Procesamiento de preguntas en lenguaje natural
  • Respuestas contextualizadas y relevantes
  • Manejo de errores y validación de entrada
  • Integración con OpenAI GPT

TranslateMessageView

Como estudiante, necesito poder traducir mensajes al español para mejorar mi comprensión del contenido.

  • Traducción precisa de textos
  • Soporte para múltiples idiomas
  • Respuestas rápidas y eficientes
  • Manejo de errores de traducción

ImageGenerationView

Como usuario, necesito generar imágenes relacionadas con el contenido de aprendizaje mediante descripciones textuales.

  • Generación de imágenes basada en prompts
  • URLs de imágenes generadas
  • Validación de prompts
  • Manejo de errores de generación

TextToSpeechView

Como estudiante, necesito convertir texto a voz para practicar la pronunciación y mejorar mis habilidades auditivas.

  • Conversión de texto a audio MP3
  • Selección de diferentes voces
  • Almacenamiento temporal de archivos
  • URLs de acceso a archivos de audio

TranscribeAudioView

Como estudiante, necesito transcribir mis grabaciones de audio para evaluar mi pronunciación y recibir retroalimentación.

  • Transcripción precisa de audio
  • Manejo de archivos temporales
  • Retroalimentación detallada

Beneficios Clave

  • Aprendizaje interactivo y personalizado
  • Retroalimentación inmediata
  • Práctica de pronunciación guiada
  • Generación de contenido dinámico
  • Soporte multilingüe

Implementación Técnica

AIService

Clase principal que maneja la integración con OpenAI para diferentes servicios de IA.

  • Integración con GPT-4 para chat interactivo
  • Análisis y retroalimentación estructurada
  • Transcripción de audio usando Whisper
  • Generación de voz con múltiples voces disponibles
  • Manejo de contexto y escenarios de conversación
  • Sistema de retroalimentación personalizado

ImageGenerator

Servicio especializado para la generación de imágenes usando DALL-E 3.

  • Generación de imágenes de alta calidad (1024x1024)
  • Procesamiento de prompts en lenguaje natural
  • Manejo de errores robusto
  • Integración con el API más reciente de OpenAI

Características Técnicas

Detalles de la implementación y configuración del sistema.

  • Gestión segura de API keys mediante variables de entorno
  • Sistema de logging para debugging y monitoreo
  • Manejo de archivos temporales para audio y transcripciones
  • Respuestas en formato JSON estructurado
  • Validación de entrada y manejo de errores
