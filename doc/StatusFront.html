Administrador

1. Gestión de Cursos

Crear Cursos

Formulario con los siguientes campos:

  • Nombre del curso
  • Descripción del curso
    • Editor de texto con opciones: color de letra, cursiva, negrita, párrafo, citas, encabezados, links, atrás, adelante, lista ordenada y desordenada
  • Cover (imagen del curso)
    • Crop
    • Swap
    • Delete
  • Publicar (on/off)
  • Definir nivel y categoría (campo select)
  • Puntos clave
    • Agregar punto clave
    • Eliminar punto clave

Eliminar Cursos

Se accede al formulario de creación de cursos, donde hay un botón Eliminar.

Al hacer clic en Eliminar, se muestra una alerta de confirmación:

  • Si se confirma, se elimina el curso.
  • Si se cancela, el curso no debe ser eliminado.

Consultar Cursos

Se deben listar todos los cursos en la página.

Cada curso debe mostrar:

  • Nivel
  • Categoría
  • Nombre
  • Imagen

Funcionalidades:

  • Publicar: Esconder o mostrar el curso a los estudiantes
  • Estudiantes: Ver los estudiantes inscritos en el curso
  • Lecciones: Ver las lecciones creadas dentro del curso.
  • Preview: Ver el curso como estudiante.
  • Editar: Modificar la información del curso

Editar Cursos

Se muestra el mismo formulario de creación con opción de edición.

Funcionalidades:

  • Modificar nombre, descripción, cover, nivel, categoría y puntos clave.
  • Mantener las mismas opciones de edición de texto y de imagen.

2. Gestión de Clases

Crear Clases

Botón Add Lesson o Create New Lesson.

Formulario con los siguientes campos:

  • Título de la clase
  • Descripción
    • Editor de texto con las mismas funcionalidades que el de cursos.
  • Cover con opciones:
    • Crop
    • Swap
    • Delete
  • Puntos clave dinámicos:
    • Agregar punto clave
    • Eliminar punto clave
  • Opciones de guardado:
    • Crear clase
    • Cancelar creación

Consultar Clases

Se deben listar todas las clases creadas.

Cada clase debe mostrar:

  • Número ordinal (iniciando en 0)
  • Nombre
  • Imagen

Funcionalidades:

  • Editar: Dirigir a la página de edición de tareas
  • Preview: Vista previa de la clase modo estudiante.
  • Ocultar: Ocultar la clase para los estudiantes
  • Publicar: Hacer visible la clase para los estudiantes
  • Settings: Editar descripción, imagen y puntos clave

Eliminar Clases

Botón Eliminar.

Se muestra una alerta de confirmación:

  • Si se confirma, se elimina la clase.
  • Si se cancela, la clase no debe ser eliminada.

3. Gestión de Tareas

Creación de Tareas

El administrador debe poder crear tareas desde un menú de navegación con las siguientes opciones:

  • Layout Block
  • Video Block
  • Text Block
  • Info Box
  • Gallery Layout
  • AI Chat
  • Multimedia Block
  • Knowledge Check

Para guardar la tarea, todos los formularios deben estar completos. Si hay campos vacíos, el botón de guardar no se habilita.

Toda la información debe quedar guardada en la base de datos.

Tipos de Bloques

1. Layout Block

Formulario con:

  • Título
  • Descripción
  • Cover
  • Audio
  • Script del audio
  • Speech to text
  • Tareas del bloque (con funcionalidad Drag and Drop y opción de incluir estadísticas):
    • Multiple Choice: Formulario dinámico con pregunta y respuesta, con checklist para seleccionar la respuesta correcta.
    • True/False: Formulario con campo de texto y select para elegir la opción correcta.
    • Categories: Similar a Multiple Choice sin checklist.
    • Fill in the Gaps: Formulario con campo para escribir el ejercicio y botones funcionales para agregar opciones.
    • Word Bank: Formulario con palabras adicionales para completar un ejercicio.

2. Multimedia Block

Video Block: Formulario para subir un video con título, descripción, subtítulos y opciones de tamaño (S, M, L).

Audio Block: Formulario para subir un audio con título, descripción y subtítulos.

3. Text Block

Formulario con:

  • Título
  • Descripción
  • Opciones:
    • Plain Text: Cuadro de texto con editor.
    • Info Box: Formulario con opciones de visualización:
      • Attention
      • Grammar
      • Tip

4. Gallery Layout

Formulario con:

  • Título
  • Descripción
  • Subida de imágenes con:
    • Título de la imagen
    • Descripción
    • Botón Eliminar para remover la imagen

5. AI Chat (Scenarios)

Formulario con:

  • 1. Subida de imagen de fondo
  • (crop, eliminar, cambiar, IA generativa).
  • 2. Goals and Objectives: Campos de texto.
  • 3. Información sobre el estudiante (nivel y limitaciones de lenguaje).
  • 4. Rol de ChatGPT.
  • 5. Rol del estudiante.
  • 6. Conversation Starter.
  • 7. Flujo de conversación:
    • Uso de vocabulario objetivo.
    • Respuestas naturales a preguntas.
  • 8. Cierre de conversación.
  • 9. Evaluación y feedback.
  • 10. Cálculo de puntaje sobre 100.
  • 11. Agregar prompts adicionales.

Funcionalidad de paginación:

  • Atrás
  • Siguiente
  • Botón Crear Escenario

6. Knowledge Check

Campos comunes:

  • Título
  • Descripción
  • Tipos de ejercicios:
    • Multiple Choice
    • True/False
    • Categories
    • Fill in the Gaps
    • Word Bank

Administración de Contenido y Tareas

Visualización del Contenido Creado

El administrador debe poder visualizar el contenido creado en la plataforma.

Consulta de Tareas en Modo Profesor

El administrador debe poder ver las tareas creadas con sus respuestas correctas y su formato correspondiente:

  • Multiple Choice: Visualizar la tarea con las respuestas correctas.
  • Info Box Tip: Cuadro de texto verde con el formato de texto creado.
  • Info Box Grammar: Cuadro de texto azul con el formato de texto creado.
  • Info Box Attention: Cuadro de texto rojo con el formato de texto creado.
  • True or False: Ver la tarea con las respuestas correctas.
  • Fill in the Gaps: Ver la tarea con las respuestas correctas.
  • Categories: Ver la tarea con las respuestas correctas.
  • Word Bank: Ver la tarea con las respuestas correctas.
  • Video: Mostrar el video sobre el background.
  • Audio: Mostrar el audio asociado.
  • Scenarios:
    • Listar los escenarios creados, visualizados como tarjetas.
    • Desde clase: Escenario específico de la clase.
    • Desde el panel del profesor (opción "Scenario"): Se listan todos los escenarios en general con los siguientes datos: nombre, categoría, nivel e imagen.

Funcionalidades Adicionales

Editar Escenario: Permite editar los elementos del formulario inicial

Publicar/Ocultar: Permite mostrar u ocultar el escenario a los estudiantes

Preview: Ver el escenario en modo estudiante (según especificaciones del rol de estudiante)

Text Block Column: Mostrar el texto creado

Gallery Layout: Mostrar imagen con título y descripción (3 por fila)

Layout Block: Si la tarea es de tipo "Layout Block", debe visualizarse con imagen, audio o ambos

Actualización de Tareas

El administrador debe poder actualizar cualquier tarea mediante las siguientes opciones:

  • Edit Block: Mostrar el formulario correspondiente para editar y actualizar la tarea
  • Copy Block: Copiar un bloque para ser pegado en otra clase
  • Move To: Mover un bloque a otra clase
  • Delete Block: Eliminar la tarea
  • Crear Tareas Entre Bloques: Permitir la creación de nuevas tareas dentro de bloques
  • Drag and Drop: Reposicionar las tareas mediante arrastrar y soltar

Visualización de Estudiantes

El administrador debe poder visualizar la lista de estudiantes sin restricciones por curso.

Lista de Estudiantes:

  • Nombre.
  • Fecha de inicio (capturada al momento de la creación del estudiante).
  • Average Score: Puntaje conglomerado según el avance del estudiante.
  • Bloquear Estudiante: Limitar el acceso sin eliminarlo de la base de datos.
  • Eliminar Estudiante: Eliminar estudiante de la base de datos.
  • Ver Información del Estudiante:
    • Nombre, teléfono, correo, avatar, niveles realizados, estado (activo/inactivo).
    • Ajustes: Redirige a la página de estadísticas del estudiante.

Creación de Usuario Administrador

1. Formulario con los siguientes campos:

  • Correo.
  • Nombre.
  • Contraseña.
  • Código de registro.

2. El sistema proporcionará el código de registro e invalidará su uso una vez se registre.

Login Administrador

Formulario con los campos correo y contraseña.

Si el usuario no está autenticado, se redirige a la página de inicio de sesión o landing page.

Si el usuario no existe, se muestra un mensaje para contactar al personal de Polyacademy.

Si la contraseña es errónea, se muestra un mensaje informativo indicando que no es válida.

Recuperación de Contraseña

Recuperación de Contraseña

1. Opcion Recuperar Contraseña.

2. El usuario proporciona su correo registrado en la plataforma.

3. La plataforma envía un correo electrónico con un enlace temporal para cambiar la contraseña.

4. Formulario de cambio de contraseña

  • Ingresar la nueva contraseña
  • Confirmar la nueva contraseña
  • Opción Guardar

5. Mensajes del sistema

La plataforma muestra un mensaje informativo sobre el cambio de contraseña.

Si es exitoso, redirige al login.

2. Dashboard Principal

Vista General

  • Cursos y clases activos:
    • Nombre, imagen del curso
    • Funcionalidades:
      • Cambiar curso
      • Ver detalles del curso
      • Mostrar ID de la clase en curso
      • Esconder curso
      • Marcar como finalizada
      • Continuar con última clase
  • Visualización de clases (solo para estudiantes inscritos):
    • Un estudiante solo puede ver sus propias clases
    • Datos mostrados:
      • Número ordinal
      • Nombre de la clase
      • Imagen y descripción
      • Indicador de estado: Completado / En curso / Sin empezar
      • Opción para repetir clase
  • Entrar a una clase:
    • Popup de información:
      • Título, descripción, imagen, palabras clave
    • Funcionalidades:
      • Botón Atrás
      • Empezar curso (redirige a la página de tareas)

Tareas

Las tareas deben tener puntuación si se incluyen las calificaciones por el profesor

  • Multiple Choice: Selección de una o varias opciones correctas
  • True/False
  • Categories: Drag and Drop para categorizar según el ejercicio
  • Fill in the gaps: Completar frases con palabras correctas
  • Word Bank: Arrastrar palabras correctas al párrafo correspondiente
  • Videos y Audios: Reproducción con fondo
  • Galería de Imágenes: Título y contraseña
  • Cuadros Informativos:
    • Grammar Box (azul)
    • Tip Box (verde)
    • Attention Box (rojo)
  • Text Column: Texto en una sola columna
  • Scenarios:
    • Text-to-Speech y Speech-to-Text
    • Enviar mensajes escritos
    • Explicación de la actividad
    • Start Conversation:
      • Imagen de fondo del profesor cubriendo toda la pantalla
      • Avatar del profesor simulando diálogo
      • Mensaje inicial con título, descripción y botón de inicio
      • Mensajes de Polly:
        • Guardar notas
        • Traducción
        • Repetir frase en voz alta
      • Opciones del estudiante:
        • "Ayúdame a responder" (sugerencias IA)
        • Enviar mensaje o grabar voz
    • Finalización:
      • Feedback en pronunciación, fluidez, gramática, respuestas largas y errores
      • Botones: Terminar, Repetir, Continuar

Métricas de Aprendizaje
